PROJECT

2023年7月7日 更新

タッチスキャンできる非接触型ICチップをカードの両面に埋め込める特許を取得しました。この新しい両面カードを利用した市場の開拓、量産、販売を支援していただける企業を探しています。

  • 中小企業
  • 製品共同開発
  • リソース提供（既存技術の提供・特許流用の検討など）
  • 1年以内
  • スタートアップ

株式会社◯◯◯◯

青森県

半導体、ソフトウェア

自社の特徴

弊社は2013年に創業、クラウドサービスを活用したスマートフォンアプリを開発、国内・海外市場へ向けてリリースしています。アプリビジネスに加え、2018年、カードの両面に非接触型ICチップを埋め込む特許を取得、この量産・販売を目標としています。一昨年、中小企業庁の「ものづくり補助金」でこの両面カードとリーダーを試作し、現在はスマホアプリでカードの両面を読むアプリを開発、カードの改良を重ねています。

解決したい課題

・カードを読み込むリーダーは基本的にIoTデバイスもしくはスマートフォンを想定しているため、クラウドサーバーとのインターフェースしたソリューションビジネスを目標としています
・国内特許を取得してすぐに国際特許取得のためのPCT出願の手続きを行いました。この段階で国際予備審査が行われましたが同種の特許はないと報告を受けています。海外市場への進出も含めた取り組みを検討していただければと考えています。

共創で実現したいこと

・非接触型ICチップを両面に埋め込んだカードを量産し、国内・海外で販売を行いたい（1枚のカードで2種類の用途）
例えば、、、
　-海外で多くのカードに組み込まれているNFCチップと国内でメジャーなFelicaチップを裏表に組み込みグローバルなカードを提案
　-クレジットカードの手書きサインをICチップ内に登録、店頭での商品購入時にタブレットへの電子サインとのマッチングを行う
　-QR決済の時のチャージの際、電子サインをスマホで行うことで、カードのICチップの電子サインとマッチングをクラウドサーバー上で行い、本人確認をする（AIサービスによるサインのマッチングを想定）
　-2種類のNFCカード（入館カード、タイムカード）を1枚のカードにまとめる 　-表は診察券＋裏はお薬手帳 (=> パーソナルヘルスケア)
　-ドアに取り付けた２つのスマートロックをカードの裏表を使って、施錠・解錠 　-二人が一度に改札口を通過できるシステム（表は介護する人、裏は介護を受ける人など）
　etc...
